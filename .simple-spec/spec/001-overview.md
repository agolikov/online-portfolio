# Project Overview

> Last updated: 2026-05-05
> Status: active

**Name:** Online Portfolio
**Description:** A full-stack personal portfolio and resume application for presenting professional experience, projects, education, certificates, skills, behavioral stories, and role-specific cover letters. Public visitors can view a default portfolio or hash-based resume pages, while the owner can manage resume variants, generate or edit cover letters, and use an AI assistant to answer questions or update selected portfolio data.
**Stage:** alpha
**Source path:** `src/` for the Vite React frontend, `server/` for the Express API and database layer.
